# DESIGN.md

## 🎨 デザインコンセプト
- **"Essentialism"**: ツールとしての主張を消し、テキストを美しく見せることに特化。
- **"Fluidity"**: 画面幅に応じて、サイドバー、リスト、エディタを動的に再配置（1〜3カラム）。

## 📱 レイアウト・コンポーネント

### 1. 3カラム・レスポンシブ・グリッド
* **Desktop (1024px〜)**: 
    - Left: ナビゲーション（タグ、ゴミ箱、設定）
    - Middle: ノートリスト（検索窓、サマリー）
    - Right: エディタ（メイン執筆エリア）
* **Tablet (768px〜1023px)**: サイドバーをドロワーへ格納し、リストとエディタの2カラム。
* **Mobile (〜767px)**: リストとエディタをスタック形式で切り替え。

### 2. エディタ UI
* **Typography**: Noto Sans または JetBrains Mono を選択可能。行間 1.6、サイズ 16px を基準とする。
* **Checkboxes**: Markdown の `- [ ]` 記法を検知し、インタラクティブなトグルを表示。
* **Status Bar**: 画面最下部に「文字数」「保存ステータス」「同期完了時刻」を表示。

### 3. ダークモード
* システム設定に連動。
* 背景 `#121212`、テキスト `#E5E5E5`、アクセント `#0A84FF`。
